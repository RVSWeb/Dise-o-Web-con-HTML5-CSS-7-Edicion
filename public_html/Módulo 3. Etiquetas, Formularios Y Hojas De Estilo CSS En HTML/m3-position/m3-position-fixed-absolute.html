The position property

The Parent4 element has position: fixed, and will NOT remain in the natural flow of the page.
It will position itself according to the viewport.
It will also act as anchor point for the absolutely positioned Child4 element:

La capa Child4 se mueve junto con la capa Parent4
Parent4 es el punto de referencia de Child4 para moverse por la pantalla

Parent4: position: fixed, bottom: 0, left: 0, right: 0 , top: 100px;
Child4: position: absolute, right: 15px, top: 70px
